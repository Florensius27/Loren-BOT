## Simple Whatsapp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/Florensius27/Loren-BOT
> cd Loren-BOT
> npm install
```
#### Run
```bash
> node . [<session name>] (session name is optional)
```
## [Y/n] = Y
 ## [Default=N] = Y

### FOR WINDOWS/VPS/RDP USER

```bash
> cd Loren-BOT

```

```bash
> node index.js
```
